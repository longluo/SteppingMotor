# SteppingMotor


A Stepping Motor project.

### 一个步进电机的项目，包括步进电机的详细原理介绍（包括如何旋转、励磁方式、相位方法），及步进电机的测试工具。


`使用主频为16M的AT89S52芯片为主控芯片，驱动芯片为STK672-610。`
`主芯片负责显示屏的驱动，UI交互，按键扫描，最关键的是产生指定频率的脉冲波实现控制步进电机的作用。`
